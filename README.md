Пояснения / примечания:
	•	purchase_log.txt здесь ожидается в формате JSON lines — по одной JSON-структуре на строку (как в вашем первом задании). Если формат другой — пришлите пример строки, и я подправлю парсер.
	•	csv.DictReader читает построчно, поэтому visit_log.csv не будет загружаться полностью в память.
	•	Открываем CSV-файлы с newline='' и encoding='utf-8' для корректной работы с csv-модулем и русским текстом.
	•	Результирующий funnel.csv будет содержать заголовок user_id,source,category и только те визиты, для которых найдена категория покупки.
